# Sporta GM Dashboard

Sistema de gestión ejecutiva para la Gerente General de Sporta Carretera a El Salvador.

## Incluye
- Evaluación Board of Directors (10 dimensiones con scores interactivos)
- Checklist diario con persistencia (se guarda automáticamente)
- CEO Scorecard semanal con semáforos y notas
- Rutina semanal Lunes–Viernes con bloques de tiempo
- Plan de crecimiento personal 90 días con checklist
- Start · Stop · Continue
- Puntos ciegos y señales de cultura

## Stack
- Next.js 14
- TypeScript
- Tailwind CSS

## Instrucciones para correr localmente

```bash
npm install
npm run dev
```

Abre http://localhost:3000
